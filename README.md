# capstone-koring-personal
캡스톤 디자인 팀 프로젝트(Koring) 개인 기여 정리용 저장소

##1. 프로젝트 소개
    배화여자대학교 캡스톤 디자인 과제로 진행한 팀 프로젝트의 일부입니다.
    관광 정보 제공을 목표로 하는 웹 서비스를 개발하였으며 본 저장소는 해당 프로젝트 중 
    본인이 담당한 기능을 정리한 개인 기여용 저장소입니다.

##2. 사용기술
    Frontend: HTML, css, JavaScript
    Backend: Python, Flask
    Database: MySQL

##3. 담당 역할

### 게시판 기능 (Frontend 담당)
    - 게시판 UI(Html / CSS / JavaScript)
    - 게시글 목록, 상세, 작성 기능 구현
    - 댓글 작성 기능 구현
    - 백엔드 API 연동을 통한 화면 테스트 및 동작 확인
    
### 지도 기능 (Kakao Map API 활용)
    - Kakao Map API를 활용한 장소 추천 페이지 지도 연동
    - 명소 페이지에서 선택된 명소 좌표 기반 지도 및 마커 표시

### 기타 UI 구현
    - 장소 추천 / 맛집 추천 / 통합 페이지 UI 구현

### 테스트 및 연동 작업
    - 장소 추천 / 게시판 더미 데이터 테스트
    - 명소 페이지 Kakao Map API 연동을 위한 Backend API 호출 및 연결 작업

###  팀장 역할
    - 팀 내 역할 분담 및 프로젝트 진행 상황 관리
    - 팀 내 협업 문서 취합·구조 정리
    - 중간 및 최종 발표 자료 준비 및 발표 담당
